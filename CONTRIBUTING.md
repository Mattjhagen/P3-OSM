# Contributing to P3 Lending Protocol

Thank you for your interest in contributing to P3 Lending! We welcome contributions from the community to help make decentralized lending more accessible and secure.

## How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issues tracker to report bugs.
- Provide a clear description of the issue and steps to reproduce.

### 2. Feature Requests
- Open an issue to discuss new features or enhancements.
- Well-documented proposals are more likely to be considered.

### 3. Pull Requests
- Fork the repository.
- Create a new branch for your changes (`git checkout -b feature/amazing-feature`).
- Ensure your code follows the existing style and passes all tests.
- Run `just ci` to verify your changes locally.
- Submit a pull request with a clear description of the change.
