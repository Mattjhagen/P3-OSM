Getting Started with P3

P3 is an experimental reputation-based lending protocol. This page explains the basics before you connect a wallet or deposit funds.

What is P3?

P3 uses a reputation score to summarize how reliably someone has met past commitments. Borrowers can build a portable track record; lenders can see a plain-language signal of risk.

Three steps to get started

  1. Create an account at p3lending.space.
  2. Complete basic verification if prompted in your region.
  3. Explore borrowing and lending options using small amounts first.
