Reputation Scoring, Phase 1

Published 2026-02-27 · Tags: reputation, product

The first iteration of the P3 reputation score focuses on transparency and safe defaults.

Scores are returned on a 0–1000 scale with components for trust, risk, and capacity. We also ship guardrails that cap scores in cases like new accounts with little data or recent serious issues.

This is an evolving model. We plan to refine it based on data, research, and feedback from both borrowers and lenders.

For details about how scores appear in the consumer app, read the Reputation score overview. For API access, see developers.p3lending.space.
