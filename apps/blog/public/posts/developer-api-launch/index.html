Developer API Launch

Published 2026-02-27 · Tags: developers, api, reputation

Today we are shipping the first version of the P3 Developer API. It exposes reputation scores through scoped API keys designed for B2B and platform integrations.

Partners can now:

  • Create API keys from the P3 portal.
  • Call /api/v1/reputation/score and related endpoints.
  • See usage and audit events per key.

For full technical details, see developers.p3lending.space and the OpenAPI spec.
